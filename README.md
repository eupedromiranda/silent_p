# silent_p

um pequeno experimento de terminal inspirado em estética de horror analógico.

## o que é

silent_p é um programa simples que roda no terminal e, na maior parte do tempo, não faz quase nada.

de vez em quando, ele emite mensagens curtas e estranhas que sugerem uma “presença” no sistema.

## como funciona

- roda em loop no terminal
- exibe “...” na maior parte do tempo
- raramente mostra mensagens aleatórias
- algumas mensagens aparecem em cores diferentes


## instalação necessária

```bash
pip install colorama
```

## execução
```bash
python main.py
```
